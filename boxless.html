Boxless

Web app avialable at:

boxlessmusic.com

Description

Web app that plays a random song based on user-specified options. The options include Billboard rank range, date range, and genre. The backend api performs a webscrape to obtain Billboard and YouTube data. Data is cached into database to increase performance.
